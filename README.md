# The Sandlot Adventure

The Sandlot Adventure is a text-based game developed in Python. Players navigate between different locations, collect items, and work toward completing the game's objective.

This project demonstrates foundational programming concepts including dictionaries, functions, loops, conditional statements, user input, and structured application logic.

## Features

- Text-based room navigation
- Player movement between connected locations
- Item collection and inventory management
- Input validation
- Win and loss conditions
- Instructions displayed during gameplay

## Technologies Used

- Python
- Dictionaries
- Functions
- Loops
- Conditional statements
- User input
- String manipulation

## Running the Program

### Requirements

- Python 3 installed on your computer
- A Python-compatible IDE or terminal

### Instructions

1. Download or clone this repository.
2. Open the project folder.
3. Run the following command:

```bash
python sandlot_adventure.py
```

Windows users may also run:

```bash
py sandlot_adventure.py
```

4. Follow the instructions displayed in the game.
5. Enter movement and item commands when prompted.
